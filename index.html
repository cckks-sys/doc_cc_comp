# Notify

点击下面按钮查看弹窗效果

# 示例

<template>
    <cc-Notifys></cc-Notifys>
    <cc-button type='primary' size="small" @click="addNotification('info','this is a info message','info')">Info Notify</cc-button>
    <cc-button type='success' size="small" @click="addNotification('success','this is a success message','success')">Success Notify</cc-button>
    <cc-button type='warning' size="small" @click="addNotification('warning','this is a warning message','warning')">Warning Notify</cc-button>
    <cc-button type='danger' size="small" @click="addNotification('error','this is a message','error')">Error Notify</cc-button>
</template>
<script lang="ts">
import { defineComponent } from 'vue';
export default defineComponent({
    methods: {
        addNotification(title: String, message: String, type: String = 'info') {
            this.$addNotification({ title: title, message, type });
        }
    }
})
</script>

 









 





将标签<cc-Notifys></cc-Notifys>引入,然后可以使用全局注册的$addNotification,添加通知

this.$addNotification({ title: "234", message: "message"});

$addNotification传入一个对象，对象参数如下

属性名称类型默认
title标题string
message提示内容string
type提示类型string，可选'info','error','success','warning''info'
